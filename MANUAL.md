## 1. 서비스 개요

옆집디노 AI 세일즈 시스템은 **네트워크 마케팅, 보험, 방문판매** 등 영업 종사자를 위한 무료 AI 도구 모음입니다.

### 포함된 앱 (총 7개)

| 앱 | 경로 | 한줄 설명 |
|---|---|---|
| 5분법칙 플래너 | `/planner` | 5분 단위 타임블록킹으로 하루 설계 |
| AI 세일즈 코치 | `/chatbot` | AI가 실시간 영업 코칭 & 스크립트 생성 |
| 영업 스타일 진단 | `/quiz` | 10가지 질문으로 나의 영업 DNA 진단 |
| 세일즈 다이어리 | `/diary` | 고객 관리 + 영업 활동 기록 + AI 코칭 |
| 수입 시뮬레이터 | `/simulator` | 목표 수입 역산 + AI 실행 플랜 |
| AI 디지털 명함 | `/card` | QR코드로 바로 공유하는 디지털 명함 |
| 메인 대시보드 | `/main` | 모든 도구를 한 곳에서 관리 |

### 기술 특징
- **100% 무료** — 모든 기능 제한 없이 사용
- **PWA 지원** — 앱처럼 홈 화면에 설치 가능
- **오프라인 사용** — 인터넷 없이도 기본 기능 사용 가능
- **데이터 안전** — 개인 데이터는 내 기기에만 저장 (localStorage)

## 4. 5분법칙 플래너

**URL**: https://app.wellstar.life/planner

매일 아침 5분 만에 하루를 설계하는 타임블록킹 플래너입니다.

### 3단계 플래닝 프로세스

#### Step 1: 브레인덤프
- 오늘 해야 할 일을 생각나는 대로 입력
- 음성 입력 지원 (마이크 버튼)
- 제한 없이 자유롭게 쏟아내기

#### Step 2: 우선순위
- 입력한 할 일들을 중요도 순으로 정렬
- 드래그하여 순서 변경
- 가장 중요한 일이 위에 오도록 배치

#### Step 3: 타임블록
- 시간대별로 할 일 배치
- 5분/15분/30분/60분 블록 선택 (레벨에 따라)
- 완료 시 체크 표시

### 레벨 시스템
| 레벨 | 블록 단위 | 조건 |
|---|---|---|
| Lv.1 입문 | 60분 | 시작 |
| Lv.2 성장 | 30분 | 연속 3일 |
| Lv.3 숙련 | 15분 | 연속 7일 |
| Lv.4 마스터 | 5분 | 연속 14일 |

### 추가 기능
- **통계**: 연속 사용일수, 완료율, 주간 차트
- **템플릿**: 영업사원, 직장인, 프리랜서, 학생, 창업가 (5가지)
- **다크/라이트 모드**: 설정에서 전환
- **글자 크기**: 소/중/대 조절
- **데이터 백업/복원**: 설정 → 내보내기/가져오기

### 설정 메뉴
- 하단 탭에서 ⚙️ 설정 탭 선택
- 다크 모드, 알림, 글자 크기 조절
- 이메일 등록, 데이터 백업/복원
- 전체 초기화 (주의: 모든 데이터 삭제)

## 5. AI 세일즈 코치 (챗봇)

**URL**: https://app.wellstar.life/chatbot

AI가 실시간으로 영업 코칭을 해주는 대화형 도구입니다.

### 6가지 시나리오 모드

| 모드 | 설명 | 추천 상황 |
|---|---|---|
| 일반 상담 | 영업 전반에 대한 조언 | 영업 고민이 있을 때 |
| 마인드셋 | 멘탈 관리 & 동기부여 | 슬럼프, 자신감 부족 |
| 라포 형성 | 고객과의 관계 구축 | 첫 만남, 신뢰 쌓기 |
| 반론 처리 | 고객 거절 대응법 | "비싸요", "생각해볼게요" |
| 클로징 | 계약 마무리 기술 | 결정 유도, 마무리 |
| 팀 관리 | 조직 운영 & 리더십 | 팀 빌딩, 교육 |

### 사용 방법
1. 상단의 시나리오 칩 중 원하는 모드 선택
2. 텍스트 입력 또는 마이크 버튼으로 음성 입력
3. AI가 맞춤형 코칭을 제공
4. 빠른 답변 버튼으로 대화 이어가기

### 대화 기록
- 자동 저장 (최대 50개 대화)
- 좌측 히스토리 패널에서 과거 대화 확인
- 새 대화 시작: 상단 "+" 버튼

### 피드백
- 각 답변 하단의 👍/👎 버튼으로 평가
- 피드백은 서비스 개선에 활용

### 참고
- AI 답변은 "30초 만에 YES" 도서의 지식을 기반으로 합니다
- 인터넷 연결 필요 (AI 답변 생성)
- 답변 생성에 2~5초 소요

## 6. 영업 스타일 진단 (퀴즈)

**URL**: https://app.wellstar.life/quiz

10가지 질문으로 나의 영업 DNA를 진단합니다.

### 진행 방법
1. 시작 화면에서 "진단 시작" 클릭
2. 10개의 질문에 4가지 선택지 중 하나 선택
3. 결과 화면에서 나의 영업 유형 확인

### 8가지 영업 유형
각 유형별로 강점, 성장 포인트, 추천 전략을 안내합니다.

### 결과 활용
- **공유하기**: 결과를 카카오톡, SNS 등으로 공유
- **이메일 리포트**: 이메일 입력 시 상세 분석 리포트 발송
- **추천 도구**: 나의 유형에 맞는 도구 추천

### 팁
- 정답이 없으니 직감대로 선택하세요
- 결과를 팀원과 비교하면 팀 역할 분배에 도움됩니다
- 시간이 지나면 다시 진단해보세요 (성장을 확인할 수 있어요)

## 7. 세일즈 다이어리

**URL**: https://app.wellstar.life/diary

고객 관리, 영업 활동 기록, AI 코칭을 한 곳에서.

### 5개 탭 구성

#### 홈 탭
- 오늘의 영업 요약
- 긴급 팔로업 알림 (D-day 기준)
- 생일 축하 알림

#### 고객 탭 (파이프라인)
5단계 칸반 보드로 고객 관리:

| 단계 | 설명 |
|---|---|
| 잠재 고객 | 아직 접촉하지 않은 고객 |
| 접촉 | 첫 연락 완료 |
| 미팅 | 대면/비대면 미팅 진행 |
| 딜 | 제안/협상 단계 |
| 휴면 | 장기 미접촉 고객 |

- 고객 추가: "+ 고객 추가" 버튼
- 입력 항목: 이름, 전화번호, 메모, 단계, 팔로업 날짜, 생일
- 단계 변경: 고객 카드에서 단계 수정

#### 활동 탭
- 매일 영업 활동 체크
- 기본 항목: 신규 연락, 팔로업, 미팅, 제안, 계약
- 커스텀 항목 추가 가능 (최대 10개)
- 일일 메모 작성

#### 리뷰 탭
- 주간 활동 통계 (막대 그래프)
- 거절 일기 + AI 코칭
  - 거절 상황과 감정을 기록하면 AI가 조언
  - "AI 코칭 받기" 버튼 클릭

#### 설정 탭
- 테마 (다크/라이트)
- 글자 크기
- 알림 시간 설정
- 데이터 백업/복원

### 데이터 저장
- 모든 데이터는 기기의 localStorage에 저장
- 다른 기기로 이동 시: 설정 → 백업 → JSON 파일 내보내기 → 새 기기에서 복원

## 8. 수입 시뮬레이터

**URL**: https://app.wellstar.life/simulator

목표 수입을 역산하여 구체적인 실행 계획을 세웁니다.

### 2가지 모드

#### 시뮬레이션 모드
- 업종 선택: 네트워크 마케팅, 보험, 방문판매, 직접 입력
- 슬라이더로 변수 조절 (팀 규모, 수수료율, 판매량 등)
- 실시간으로 예상 수입 계산

#### 목표 역산 모드
- 목표 금액 입력 (예: 월 300만원)
- 업종 선택
- AI가 달성을 위한 구체적 실행 계획 생성:
  - 필요한 고객 수
  - 일일/주간 활동량
  - 단계별 타임라인

### AI 플랜 생성
- "AI 실행 플랜 생성" 버튼 클릭
- 인터넷 연결 시: AI가 맞춤형 플랜 생성
- 오프라인 시: 로컬 기반 기본 플랜 제공 (토스트 알림 표시)

### 다이어리 연동
- 생성된 플랜을 다이어리에 저장 가능
- "다이어리에 저장" 버튼 클릭

## 9. AI 디지털 명함

**URL**: https://app.wellstar.life/card

전문적인 디지털 명함을 만들어 QR코드로 공유합니다.

### 명함 만들기

#### 정보 입력 (왼쪽 탭)
- **필수**: 이름, 직책/타이틀
- **선택**: 전화번호, 이메일, 자기소개
- **프로필 사진**: 사진 선택 (2MB 이하)
- **SNS 링크**: Instagram, Kakao, YouTube, Blog

#### 테마 선택 (오른쪽 탭)
- 6가지 그라데이션 테마
- 실시간 미리보기

### QR 코드
- 명함 정보가 vCard 형식으로 QR 코드에 담김
- 상대방이 QR 스캔 시 연락처에 바로 저장 가능
- "QR 저장" 버튼으로 이미지 다운로드

### 공유하기
- 공유 버튼 클릭 → 카카오톡, 문자, SNS 등으로 전송
- 클립보드 복사 지원

### 팁
- 프로필 사진은 정사각형이 가장 잘 나옵니다
- 자기소개는 2~3줄이 적당합니다
- SNS 링크는 입력한 것만 명함에 표시됩니다

## 11. PWA 설치 방법

각 앱을 스마트폰 홈 화면에 설치하면 앱처럼 사용할 수 있습니다.

### iPhone (Safari)
1. 앱 페이지 접속
2. 하단 공유 버튼 (⬆️) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭

### Android (Chrome)
1. 앱 페이지 접속
2. 상단 "설치" 배너 클릭, 또는
3. 주소창 오른쪽 ⋮ 메뉴 → "앱 설치" 또는 "홈 화면에 추가"

### 설치 추천 앱
- **5분법칙 플래너** — 매일 아침 바로 열어 사용
- **메인 대시보드** — 모든 도구를 한 곳에서

## 12. 데이터 관리

### 저장 방식
- 모든 사용자 데이터는 **기기의 브라우저(localStorage)**에 저장
- 서버로 전송되지 않음 (개인정보 안전)
- 단, AI 챗봇 대화 시 질문 내용은 AI 서버로 전송되어 답변 생성에 사용

### 데이터 백업
- 플래너: 설정 → 내보내기
- 다이어리: 설정 → 백업

### 주의사항
- 브라우저 캐시/쿠키 삭제 시 데이터가 함께 삭제될 수 있음
- 시크릿/프라이빗 모드에서는 데이터가 저장되지 않음
- 기기 변경 시 반드시 백업 → 복원 진행
- "전체 초기화"는 복구 불가능

## 13. 자주 묻는 질문 (FAQ)

### Q. 회원가입이 필요한가요?
**A.** 아니요. 모든 도구를 회원가입 없이 바로 사용할 수 있습니다.

### Q. 무료인가요?
**A.** 네, 모든 기능이 100% 무료입니다.

### Q. 내 데이터는 안전한가요?
**A.** 모든 데이터는 내 기기의 브라우저에만 저장됩니다. AI 챗봇 사용 시에만 질문 내용이 AI 답변 생성을 위해 서버로 전송됩니다.

### Q. 오프라인에서도 쓸 수 있나요?
**A.** 플래너, 다이어리, 명함 등 대부분의 기능은 오프라인에서도 사용 가능합니다. AI 코칭, 챗봇 등 AI 기능은 인터넷 연결이 필요합니다.

### Q. 다른 기기에서 사용하려면?
**A.** 현재 기기에서 데이터 백업(JSON 파일) → 새 기기에서 복원하면 됩니다.

### Q. 챗봇이 답변하지 않아요.
**A.** 인터넷 연결을 확인해주세요. 서버가 바쁠 때는 잠시 후 다시 시도하면 됩니다.

### Q. 플래너 데이터가 사라졌어요.
**A.** 브라우저 캐시를 삭제하면 데이터도 삭제됩니다. 정기적으로 백업을 권장합니다.
